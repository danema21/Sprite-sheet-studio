Toolbar:

In the toolbar you can use various tools for drawing in the canvas, like a pencil(to draw pixels based on the scale of the grid), a brush(to just draw freely), an eraser, a color fill tool, an eyedropper(to copy a color previusly used on the canvas), a drag tool(to reposition the canvas in the work space), a copy tool(that copies an entire frame and on the next click it pastes it in the frame of your choice), and a stamp tool(it works the same as the copy tool but it is pasted with transparency in the grid not the canvas; it works great as a guide to your next frame drawing), also when you copy something with the copy or the stamp tool, you can cancel the action by using the right click. Then you have the usual default colors palette and a custom color picker.

In the next section is the grid controllers, here you can toggle the grid on and off and also specify how many columns and rows your frame will have(is important that you know, once you add a new frame the grid controllers section will be inhabilitated).

Lastly the animation controllers section, here you can add and remove the animation frames and rows.

Current frame bar:

This bar is useful when you want to go to a specific frame in your an animation that has a lot of frames, so you can click for example in the number 42 of the frame bar and it will take you to that frame in the canvas.

Menu options:

When clicking on New you will be asked to choose a scale for your new project, this means that you will specify of big you want the squares of the grid to be.

When you save your project it is best to name it like: example8x7-16, that means that the project consist of 8 grid columns and 7 grid rows, whit a scale of 16(the columns and rows are just a way for you to remember the width and height of the animation frame, bust the scale number is nessesary or the grid wont work very well).

Remember the weird name we gave our saved project?, well now we will put it to use. Here it ask for a width and a height, if we want to open the project example8x7-16 the we fill the field width = 8 and height = 7, and then we select the file example8x7-16.

And the last one is Animation preview, when you click on this you will see an Animation frames field(where you will enter the number of frames you want in the animation), an Animation row field(where you can choose wich row of frames will be played), and an Animation speed field(the greater the number the faster).
